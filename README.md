# Morse Code Encoder & Decoder

A C-based Morse Code application that converts text into Morse code and Morse code back into text.

## Features

* Text to Morse code conversion
* Morse code to text conversion
* Binary tree based Morse decoding
* Character statistics tracking
* Command-line support
* Interactive menu interface
* File-based statistics logging

## Technologies Used

* C Programming
* Binary Trees
* Dynamic Memory Allocation
* File Handling
* Command-Line Arguments

## Learning Outcomes

This project strengthened my understanding of:

* Dynamic memory management using malloc and realloc
* Binary tree construction and traversal
* File input/output
* Data structures
* Modular program design
